# 目的
企業における強化学習の用途として、投資や在庫のオペレーションの最適化が考えられる。
今回、より用途が多いと考えられる在庫を想定の上で、深層強化学習の有効性について確認する

# 使用するデータ
コカ・コーラの小売業者の在庫と、顧客からの購入の打診のデータ
在庫不足による機会損失を無くし、在庫による評価損を最小化するようなオペレーションを深層強化学習により実現する

# モデル構成
全て線形結合層で構築
